## 📌 주요 기능
- 투 두 등록
하루동안 해야할 일을 등록합니다.
중요도를 등록하면 중요한 순으로 정렬됩니다.

- 리워드 시스템
할 일을 완료하면 포도알 1개를 얻습니다.
15개의 일을 마치면 포도 한 송이가 생깁니다.

- 생산성 관리
Achievement 탭에서 그동안 모은 포도송이와 해낸 일들이 기록됩니다.
하루 평균 해낸 일의 갯수와 달성률이 표시됩니다.
